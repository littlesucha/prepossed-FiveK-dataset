# MIT-FiveK-dataset
I have created a download link for the FiveK dataset. It contains both the 480p version and the original 4K-resolution version, including the input images as well as the corresponding retouched results from Experts A, B, C, D, and E, for public download.


The MIT-FiveK package contains three folders:

1. MIT-fiveK-expertA_B_D_E-480p

This folder contains compressed files generated after downloading the original FiveK dataset and exporting the images for Experts A, B, D, and E through Lightroom. These images can be used as references.

2. MIT-fiveK-expertC-480p

This folder contains compressed files from the 480p version of the dataset provided by Zeng et al. in the Image-Adaptive-3DLUT project, including the input images and the Expert C ground truth (GT).

3. MIT-Adobe_FiveK

This folder contains the compressed files downloaded from the official MIT-Adobe FiveK dataset website, which include the following contents:

5,000 photos in DNG format
This format can be read by DCRAW, Adobe Lightroom, and many other tools.
An Adobe Lightroom catalog containing retouching results from 5 experts
This includes the values of the adjustment sliders for each photo, as well as the complete editing history.
Semantic information for each photo
Including indoor/outdoor, time of day, type of light, and the main subject (people, nature, man-made objects, etc.).

Note:
The expert retouching results included in this archive are only stored as a Lightroom catalog, not as exported images.
To obtain the actual retouched images, they still need to be processed and exported through Lightroom. However, this is quite easy, and you can ask GPT for the specific method. The export resolution can also be chosen as needed.
